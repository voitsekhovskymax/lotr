# 📚 Context7 Resources

Тут зібрані всі офіційні ідентифікатори (Library IDs) баз даних Vintage Story, які AI-агент автоматично використовує для швидкого доступу через MCP-сервер `context7`.

## 📖 Vintage Story Wiki

- **URL:** [https://context7.com/websites/wiki_vintagestory_at](https://context7.com/websites/wiki_vintagestory_at)
- **Context7 ID:** `/websites/wiki_vintagestory_at`
- **Опис:** Основна вікі гри. Використовується для загального розуміння механік гри, JSON-конфігурацій блоків, предметів, сутностей та базового моддингу.

## 🛠 Vintage Story API Docs

- **URL:** [https://context7.com/websites/apidocs_vintagestory_at](https://context7.com/websites/apidocs_vintagestory_at)
- **Context7 ID:** `/websites/apidocs_vintagestory_at`
- **Опис:** Офіційна документація по C# API. Агент використовує це джерело для пошуку методів `ICoreAPI`, інтерфейсів взаємодії з блоками, рендерингу та обробників подій.

## 🎮 VS Survival Mod (Source)

- **URL:** [https://context7.com/anegostudios/vssurvivalmod](https://context7.com/anegostudios/vssurvivalmod)
- **Context7 ID:** `/anegostudios/vssurvivalmod`
- **Опис:** Вихідний код ванільного моду Survival від розробників гри (Anego Studios). Це **найкраще джерело прикладів** реального коду для створення кастомних AI-завдань (`AiTasks`), логіки розмноження, складних багатоблокових структур та інтеграції систем.
