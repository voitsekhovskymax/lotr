     Ось покрокова інструкція:                                                                                                                                                       
                                                                                                                                                                                     
     1. Запусти новий світ                                                                                                                                                           
     Старий світ не підійде — регіони видно тільки в нових чанках. Створи новий одиночний світ.                                                                                      
                                                                                                                                                                                     
     2. Перевір лог при завантаженні                                                                                                                                                 
     В консолі сервера (або лог файлі) мають бути ці рядки:                                                                                                                          
                                                                                                                                                                                     
     [LOTR] PngBiomeMap loaded 256x256, 8 region colors, scale=512 blocks/px                                                                                                         
     [LOTR] Loaded 8 biomes, 8 regions, PNG map=True                                                                                                                                 
     [LOTR] RegionSystem: spawn cached at X=512000 Z=512000                                                                                                                          
                                                                                                                                                                                     
     Якщо PNG map=False — щось пішло не так з декодуванням.                                                                                                                          
                                                                                                                                                                                     
     3. Знайди де ти відносно спавну                                                                                                                                                 
                                                                                                                                                                                     
     /lotr region                                                                                                                                                                    
                                                                                                                                                                                     
     Покаже твої spawn-relative координати. Наразі ти, мабуть, в Wilderness (чорний піксель або суша без регіону).                                                                   
                                                                                                                                                                                     
     4. Телепортуйся в центри регіонів                                                                                                                                               
     Карта 256x256, центр пікселів = (128, 128) = spawn 0,0. Кожен піксель = 512 блоків.                                                                                             
                                                                                                                                                                                     
     За скриптом генерації центри регіонів приблизно такі (spawn-relative):                                                                                                          
                                                                                                                                                                                     
     Shire        → /tp =(spawnX-15360) 150 =(spawnZ-17920)                                                                                                                          
                     або простіше: spawn-relative X≈-15000 Z≈-18000                                                                                                                  
                                                                                                                                                                                     
     Регіони відносно spawn (pixel - 128)  512:                                                                                                                                      
       Shire        pixel ~(86, 79)  → X = (86-128)512 = -21504  Z = (79-128)*512 = -25088                                                                                           
       Bree         pixel ~(112, 95) → X = -8192         Z = -16896                                                                                                                  
       Rivendell    pixel ~(137, 83) → X = +4608         Z = -23040                                                                                                                  
       Mordor       pixel ~(184,181) → X = +28672        Z = +27136                                                                                                                  
                                                                                                                                                                                     
     Найпростіший спосіб — телепортуйся покроково і перевіряй:                                                                                                                       
                                                                                                                                                                                     
     /tp ~-21504 150 ~-25088                                                                                                                                                         
     /lotr region                                                                                                                                                                    
                                                                                                                                                                                     
     Має показати "The Shire" замість "Wilderness".                                                                                                                                  
                                                                                                                                                                                     
     5. Якщо регіон не знаходиться                                                                                                                                                   
     Скажи мені spawn X/Z зі step 3 — я порахую точні координати для твого конкретного світу. Різні світи мають різний spawn offset.
